# **App Name**: Echoes of Us

## Core Features:

- Animated Celestial Hero: A full-screen mobile hero section featuring an animated night sky background with soft stars, a romantic heading, subtext, and a 'Our Journey' button that smooth scrolls.
- Romantic Vertical Timeline: A 'Our Journey' section presenting key relationship milestones in a vertical, mobile-optimized layout with small photos, short emotional paragraphs, and fade-in animations on scroll.
- Mobile Photo Gallery: A swipeable image slider with rounded corners and soft glow borders. Users can click to expand images to full screen with smooth transition effects.
- Love Letter Writing Tool: An AI-powered tool to assist in drafting emotional long love messages, displayed in a soft glowing card with a handwritten cursive-style font. The first line features a typing animation effect.
- Embedded Romantic Music Player: Automatically plays placeholder romantic background music upon first user interaction with the page, controllable via a mute/unmute button in the transparent navigation bar.
- Glassmorphism Navigation: A fixed transparent glass-style mobile navigation bar with a heart logo, '4 Years of Us' title, and music controls.
- Dreamy Footer: A footer section featuring a romantic quote and a small heart animation, with a soft fade-out effect.

## Style Guidelines:

- Dark color scheme for a dreamy, night-sky aesthetic. Primary interactive color: Soft Rose Pink (#E585BA) to evoke romance and warmth. Background: Deep Plum-Grey (#2C272C) for a subtle, ethereal dark sky base. Accent: Vibrant Magenta-Purple (#CC4DBD) for highlights and contrast, enhancing the magical feel.
- Additional highlight color: Warm Gold (#FFD700) will be used for subtle glows, star elements, and borders to enhance the dreamy atmosphere, as described in 'warm golden glow highlights' from the prompt.
- Headings font: 'Playfair' (serif) for an elegant, high-contrast, romantic feel, mimicking the intended 'pixel-style romantic font' with its unique letterforms. Paragraphs font: 'Alegreya' (serif) for a soft, intellectual, and contemporary romantic cursive-style text.
- Utilize soft, glowing heart motifs and ethereal particle effects to create a floating, dreamy ambiance. A simple, elegant heart logo for navigation.
- Mobile-first, optimized for screens like the iPhone 14 Pro. Soft blur glassmorphism cards for content sections, rounded UI elements for a gentle, approachable aesthetic. All sections will be designed for seamless vertical scrolling on mobile devices.
- Implement subtle floating heart or light particle animations across the background. Incorporate smooth scrolling, fade-in animations for section content (especially the timeline), click-to-expand image animations in the gallery, and a typing animation for the love message.
